# YoordConfig V1
Auto Install Apps, Configs, ... On Your Debian Base Distro.
<p>


  &nbsp;

<b>How To Install YoordConfig</b>

1 - git clone https://github.com/Yoord1992/YoordConfig
<p>
2 - cd YoordConfig
<p>
3 - chmod +x ./start
<p>
4 - ./start
